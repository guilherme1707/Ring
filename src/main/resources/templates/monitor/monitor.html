Welcome to Ring


Ring is a tool to keep track of user generated NoSQL data,
where you can define a template, upload files and validate uploaded data.
This way Ring becomes a new source you can integrate with your tools extracting data via API,
and Ring will take care of providing data in a more fault-tolerant way to your processes.

Manual Inputs

This feature enables the creation of templates that says the pattern the user file should follow to upload in the system

Groups

It is possible to register access groups for users, that defines a set of manual input of the group and what users can manage that.
Users

This feature allows the creation of new users to the systems. it is also possible to link specific accesses for this user and / or to link them to groups.
